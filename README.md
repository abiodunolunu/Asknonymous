# Asknonymous

Asknoymous is a website where you can send and receive anonymously

## Installation

Use the package manager [npm](https://nodejs.org/) to install nodejs.

```bash
npm install
```
to install dependencies

## Usage

```bash
npm start
```
to get the app running
